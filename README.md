# Run locally

This app analyzes uploaded JPG/PNG images using OpenAI Vision via a local proxy endpoint (`/api/analyze`).

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set `OPENAI_API_KEY` in [.env.local](.env.local)
3. Run the app:
   `npm run dev`
